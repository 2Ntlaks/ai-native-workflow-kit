# Cursor Rules Template

Use this as a starting point for `.cursor/rules/project.mdc` or adapt it into Cursor project rules.

```mdc
---
description: AI-native workflow kit repository rules
alwaysApply: true
---

Use START_HERE.md to choose the workflow.
Use templates/agent-task-brief.md for delegated tasks.
Follow docs/autonomy-levels.md for autonomy boundaries.
Keep changes scoped and evidence-backed.
Do not touch secrets, auth, migrations, destructive operations, dependencies, or broad refactors without explicit approval.
End implementation work with templates/review-packet.md.
```

## Verify Before Use

Cursor rules, `.cursor/rules`, `.cursorrules`, AGENTS.md support, scoping, and rule application behavior change over time. Check current Cursor docs before standardizing a project.
