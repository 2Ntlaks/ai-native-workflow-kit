# Task Readiness

Use before giving work to an agent. If any blocker is unchecked, revise the brief first.

- [ ] Objective says what must be true when done.
- [ ] Non-goals are listed.
- [ ] Autonomy level is selected.
- [ ] Files, systems, or unknown areas are named.
- [ ] Acceptance criteria can be tested or inspected.
- [ ] Verification command or manual check is listed.
- [ ] Stop conditions cover dependencies, config, migrations, auth/security, destructive actions, and private data.
