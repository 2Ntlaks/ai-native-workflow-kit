# Human-Agent Contract

## Human Responsibilities

Define outcomes, choose autonomy, approve permissions, review plans, inspect diffs, and make final product decisions.

## Agent Responsibilities

Inspect relevant code, state assumptions, keep changes scoped, follow existing patterns, run verification, provide evidence, and stop at stop conditions.

## Stop Conditions

Stop for secrets, credentials, private data, auth, payments, migrations, destructive actions, dependency changes, broad refactors, source conflicts, or unclear verification.
