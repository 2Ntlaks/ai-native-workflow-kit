# Context Budget

Long sessions, noisy logs, repeated files, and irrelevant docs make important facts harder to find.

## Rules

- Prefer file paths over pasted full files when the agent can read them.
- Paste exact error messages, not entire terminal histories.
- Summarize previous decisions and link to decision records.
- Keep AGENTS.md concise and point to deeper docs.
- Split long work into tasks with review packets.
