# Maintenance

Update the kit when the same agent mistake happens twice, a workflow becomes routine, a tool changes behavior, a checklist misses an issue, or a source becomes stale.

## Monthly Review

1. Read recent review packets and retrospectives.
2. Scan `failures/` for repeated patterns.
3. Update one template or checklist at a time.
4. Verify links in `research/source-index.md`.
5. Delete vague or duplicated guidance.
6. Add a changelog entry.
