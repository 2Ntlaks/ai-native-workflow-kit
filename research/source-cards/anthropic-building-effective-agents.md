## Key Claims

- Use simple, composable patterns before complex frameworks.
- Workflows follow predefined paths while agents dynamically direct tools and process.
- Tool interfaces need careful design and examples.

## Caveats

- The article is about agentic systems broadly, not only coding agents.

## Workflow Implications

- Multi-agent work is not the default workflow.
- The workflow set favors simple reusable loops.
- Templates make the human-agent interface explicit.
