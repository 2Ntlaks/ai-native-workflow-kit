# Research Brief

## Question

What decision must this research help the human make?

## Decision This Research Supports

Example: choose a library, decide an architecture direction, confirm tool behavior, or identify migration risk.

## Allowed Sources

- Current repository
- Official docs
- Primary research or benchmark material
- High-quality secondary analysis only when primary sources are unavailable

## Required Claims Format

For each important claim, provide:

- Claim:
- Source:
- Caveat:
- Workflow implication:

## Required Output

- Short answer
- Source-backed claims
- Caveats
- Recommendation
- Open questions

## Stop Conditions

- Official docs conflict with repo behavior.
- A required source is unavailable.
- The answer depends on a tool capability that must be verified in the target environment.
